Student Performance Analysis

Project Overview
This project analyzes student performance using Python, Pandas, and Matplotlib.

Features
- Read student data from CSV file
- Calculate Total Marks
- Calculate Average Marks
- Find Top Performer
- Find Lowest Performer
- Generate Student Rankings
- Display Subject-wise Averages
- Visualize data using Bar Charts
- Visualize data using Pie Charts

Technologies Used
- Python
- Pandas
- Matplotlib
- CSV

Dataset

The dataset contains:
- Name
- Maths Marks
- Science Marks
- English Marks
- Telugu Marks

Output
- Student Performance Report
- Ranking Report
- Bar Chart of Total Marks
- Pie Chart of Subject-wise Distribution
